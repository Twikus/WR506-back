
# WR506 - Back - Movies API

This is an Movies API for a university project.
It was develop with Symfony.




![Logo](https://cdn.worldvectorlogo.com/logos/symfony.svg)


## Deployment

To run this project : 
- Run

```bash
  composer i
```

- Run MAMP

- Copy .env.local for .env
```bash
  cp .env.local .env
```
